Discover a Wealth of Notable Features in Sandboxie-Plus

When you make the switch to Sandboxie-Plus, you gain access to a plethora of notable features designed to enhance your sandboxing experience. Here are just a few highlights of what you can expect:

 

1.       Enhanced Security Sandboxes: Restrict syscall elevation and limit access to device endpoints, ensuring only safe and filtered operations are permitted.

 

2.       Privacy Protection: Apply preset rule collections to safeguard personal data locations, creating an empty environment within sandboxes that protects your sensitive information.

 

3.       Rule Specificity: Prioritize rules based on their specificity, allowing subpaths to be readable/writeable while parent parts remain protected.

 

4.       Compartment Mode: Optimize compatibility by using a reduced isolation scheme while still maintaining a high level of isolation for seamless application experience.

 

5.       Windows Filtering Platform (WFP) Support: Utilize Sandboxie as an application firewall, applying rules per box to manage internet access on a per-application basis.

 

6.       Snapshots: Easily revert sandboxes to defined previous states or benefit from auto-reverting to a fresh clean box with preset configurations.

 

7.       Improved User Interface: Enjoy a new, intuitive Qt-based UI with dark mode, customizable run menus, global hotkeys, and an INI section editor for advanced configuration.
